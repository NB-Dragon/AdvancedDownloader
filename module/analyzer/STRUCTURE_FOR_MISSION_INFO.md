# Structure Define
- `target_link`: URL to connect to the resource.
- `save_path`: The path to save without file name.
- `thread_count`: The maximum number of threads.
- `headers`: The headers to request.
- `proxy`: The proxy to request.

# Example
```json
{
  "target_link": "",
  "save_path": "",
  "thread_count": 1,
  "headers": {},
  "proxy": ""
}
```
